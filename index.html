Conectar la fuente de datos real

Esta app ya está preparada para API-Football mediante un backend incluido en la carpeta /server. Sigue estos pasos:

  • ⚽ API-Football — ya está conectado en el código (fixtures, cuotas, predicciones, estadísticas, lesiones, goleadores)
  • 🗂️ Sportmonks — misma arquitectura, cambiando las funciones normalize* del backend
  • 📅 Football-Data.org — alternativa para calendarios y resultados
  • 💰 The Odds API — alternativa específica para cuotas
  • 🏦 Casas de apuestas autorizadas — mediante su API oficial

Pasos para activarlo:

365Scores y Google no ofrecen una API pública que un navegador pueda consumir directamente (ni con CORS ni con términos de uso que lo permitan). Por eso este backend usa API-Football, que sí publica una API con licencia pensada para esto.
